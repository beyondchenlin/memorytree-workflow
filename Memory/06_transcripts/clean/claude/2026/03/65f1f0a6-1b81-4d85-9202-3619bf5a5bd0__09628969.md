### Heartbeat Entry Point (heartbeat.py, FULLY IMPLEMENTED)

**Main flow** (lines 39-79):
1. Load config from `~/.memorytree/config.toml` (uses defaults if missing)
2. Acquire PID-based lock from `~/.memorytree/heartbeat.lock`
3. Iterate over registered projects
4. For each: discover transcripts → parse → import → scan for sensitive info → git commit/push
5. Release lock on exit (try/finally ensures cleanup)

- Commits with message: `"memorytree(transcripts): import X transcript(s)"`
- Conditionally pushes if `auto_push=true` and remote exists
