### Generic Utilities

**slugify(value: str, fallback: str = "session") -> str**
- Converts string to slug format
- Logic: ASCII-encode string, lowercase, replace non-alphanumeric (except `.`, `-`, `_`) with `-`, strip leading/trailing `-._`
- Returns `fallback` if result is empty

**sha256_file(path: Path) -> str**
- Reads file in 1MB chunks, computes SHA256 digest
- Returns hexdigest (64 hex characters)

**normalize_timestamp(value: Any, fallback: str | None = None) -> str**
- Handles multiple input types:
  - **int/float**: Unix timestamp → converts to UTC datetime → removes microseconds → ISO format → replaces `+00:00` with `Z`
  - **datetime object**: Convert to UTC if needed → remove microseconds → ISO format with Z suffix
  - **string**: Strip whitespace, if empty use fallback or current time; if ends with `Z`, replace with `+00:00`, parse as ISO → convert to UTC → remove microseconds → ISO with Z
  - **other**: Use fallback or current time in UTC
- Fallback: `normalize_timestamp(datetime.now(timezone.utc))`

**earliest_timestamp(current: str, candidate: Any) -> str**
- Normalizes candidate, parses both as ISO timestamps
- Returns whichever is earlier (or the valid one if one is None)
- Falls back to `current` or `normalized` if one is invalid

**parse_iso_timestamp(value: str) -> datetime | None**
- Parses ISO string (handles Z suffix by converting to +00:00)
- Returns UTC datetime or None if parsing fails
- If no timezone info, assumes UTC

**timestamp_partition(timestamp: str) -> tuple[str, str]**
- Returns (year, month) strings in "%Y", "%m" format
- Falls back to ("unknown", "unknown") on parse error

**join_paragraphs(parts: Any) -> str**
- Converts all parts to strings, strips whitespace, filters empty strings
- Joins with `\n\n`

**summarize_value(value: Any, limit: int = 180) -> str**
- None → "none"
- String → collapse whitespace, truncate to limit
- Other → JSON serialize (sorted keys, no ASCII escaping), then truncate
- Truncation: if `len(text) <= limit`, return as-is; otherwise `text[:limit-3] + "..."`

**truncate(text: str, limit: int) -> str**
- Returns text if <= limit, otherwise `text[:limit-3] + "..."`

**yaml_escape(value: str) -> str**
- Escapes for YAML: `\` → `\\`, `"` → `\"`, newline → `\n`
- Wraps result in double quotes

**ensure_dict(value: Any) -> dict[str, Any]**
- Returns value if it's a dict, else empty dict

**ensure_list(value: Any) -> list[Any]**
- Returns value if it's a list, else empty list

**get_nested(payload: dict[str, Any], *keys: str) -> Any**
- Navigates nested dict with multiple keys
- Returns None if any intermediate key is not a dict

**load_jsonl(path: Path) -> list[dict[str, Any]]**
- Reads file line by line (UTF-8, ignoring decode errors)
- Each line: strip, skip if empty, parse JSON (skip on decode error)
- Returns list of all successfully parsed objects

**load_json(path: Path) -> dict[str, Any] | None**
- Returns None if path doesn't exist
- Tries to parse file as JSON (UTF-8)
- Returns None on OSError or JSONDecodeError

### Text Extraction Helpers

**extract_text_blocks(blocks: Any) -> str**
- Ensures input is list
- For each block:
  - If string: strip, add to parts if non-empty
  - If dict:
    - Get `type` field (lowercase), skip if in SKIP_BLOCK_TYPES | TOOL_USE_TYPES | TOOL_RESULT_TYPES
    - If type in TEXT_BLOCK_TYPES: get `text` field, add if non-empty
    - Otherwise: try `extract_simple_text()` fallback
- Returns `join_paragraphs(parts)`

**extract_simple_text(value: Any) -> str**
- String: return stripped
- List: recursively extract and join paragraphs
- Dict: check keys ("text", "content", "value") in order, return first string value (stripped)
- Other: return empty string

**extract_gemini_text(node: dict[str, Any]) -> str**
- Tries to extract from `parts`, `content`, `text`, `message` fields
- Calls `extract_gemini_parts()` on each and collects results
- Returns `join_paragraphs(parts)`

**extract_gemini_parts(value: Any) -> list[str]**
- String: strip, return as single-item list if non-empty
- List: recursively call on all items, collect results
- Dict:
  - Get `type` field, skip if in SKIP_BLOCK_TYPES | TOOL_USE_TYPES | TOOL_RESULT_TYPES
  - If has `text` key (string): strip, return as single-item list if non-empty
  - Otherwise: recursively extract from `parts` and `content` fields
- Other: return empty list

**find_first_mapping_with_keys(value: Any, keys: set[str]) -> dict[str, Any] | None**
- Recursively searches structure (dicts and lists) for a dict containing any of the keys
- Returns first match found, or None

### Deduplication Helpers

**_content_hash(text: str) -> str**
- Returns first 16 chars of SHA256 hexdigest of text

**deduplicate_messages(messages: list[TranscriptMessage]) -> list[TranscriptMessage]**
- Deduplicates by `(timestamp, role, content_hash)` tuple
- Keeps first occurrence, skips duplicates
- Returns deduplicated list in original order

- Keeps first occurrence, skips duplicates

### Client Inference & Parsing

**infer_client(client: str, source: Path) -> str**
- If `client != "auto"`, return client as-is
- Otherwise, check source path (case-insensitive):
  - If contains `/.codex/` or filename contains `rollout-` → "codex"
  - If contains `/.claude/` or `/projects/` → "claude"
  - If contains `/.gemini/` or filename contains `checkpoint` → "gemini"
- Raises SystemExit if cannot infer

**parse_transcript(client: str, source: Path) -> ParsedTranscript**
- Dispatch function: calls appropriate parser based on client name
- Raises SystemExit if client unsupported

### Codex Parser

**parse_codex_transcript(path: Path) -> ParsedTranscript**
1. Load JSONL records
2. Initialize: `session_id = title = path.stem`, `started_at = file mtime`, `cwd = branch = ""`
3. For each record:
   - Get `type` field (string), normalize timestamp
   - If type == "session_meta":
     - Update session_id, title, started_at (use earliest), cwd, branch from payload
     - Continue
   - Skip if type not in {"response_item", "event_msg"}
   - Get payload, check payload.type:
     - **"message"**: Extract role (must be "user"/"assistant"), text from content blocks, create TranscriptMessage
     - **"user_message"/"agent_message"**: role = "user" or "assistant", text from message field
     - **"function_call"**: name field (or "function_call"), arguments field (or input), create tool event with summary: `f"{name} input={summarize_value(arguments)}"`
     - **"custom_tool_call"**: name (or call_id or "custom_tool_call"), create tool event with input summary
     - **"function_call_output"**: name, output (or content), create tool event with summary: `f"{name} output={summarize_value(output)}"`
     - **"custom_tool_call_output"**: name, output (or content), create tool event with output summary
4. Deduplicate messages and tool events
5. Return ParsedTranscript with client="codex"

### Claude Parser

**parse_claude_transcript(path: Path) -> ParsedTranscript**
1. Load JSONL records
2. Initialize: `session_id = title = path.stem`, `started_at = file mtime`, `cwd = branch = ""`
3. For each record:
   - Extract `type`, timestamp
   - Update session_id from `sessionId`, started_at (earliest), cwd, branch fields
   - Skip if type not in {"user", "assistant"}
   - Get message dict from record.message
   - Extract role from message.role or use record_type
   - Parse content:
     - If string: check if non-empty and role valid, create TranscriptMessage
     - If list of blocks:
       - Collect text from blocks with type in TEXT_BLOCK_TYPES
       - Skip SKIP_BLOCK_TYPES
       - For TOOL_USE_TYPES: create tool event with summary `f"{name} input={summarize_value(input)}"`
       - For TOOL_RESULT_TYPES: create tool event with summary `f"{tool_use_id} output={summarize_value(content)}"`
       - Create single TranscriptMessage from collected text_parts if valid role and non-empty
4. Deduplicate messages and tool events
5. Return ParsedTranscript with client="claude"

### Gemini Parser

**parse_gemini_transcript(path: Path) -> ParsedTranscript**
1. Initialize: `session_id = title = path.stem`, `started_at = file mtime`, `cwd = branch = ""`
2. Load payload:
   - If .jsonl suffix: use `load_jsonl()`
   - Otherwise: parse as JSON string
3. Find first metadata dict containing keys {"sessionId", "chatId", "cwd", "branch", "timestamp"}
4. If found, extract session_id, started_at (earliest), cwd, branch
5. **Recursive visit(node) function**:
   - Skip non-dicts and non-lists
   - For lists: visit all items
   - For dicts:
     - Update session_id, started_at (earliest), cwd, branch from fields
     - Check for role ("user"/"assistant"/"model"), normalize "model" to "assistant"
     - Get timestamp, extract text via `extract_gemini_text(node)`
     - Create TranscriptMessage if valid role and non-empty text
     - Check for tool usage:
       - tool_name from: `node.toolUse.name` OR `node.toolName` OR `node.functionName` OR `node.tool`
       - If found: create tool event with summary `f"{tool_name} input={summarize_value(node.args or node.input)}"`
     - Recursively visit all node values
6. Call visit(payload)
7. Deduplicate messages and tool events
8. Return ParsedTranscript with client="gemini"

## Critical Implementation Notes for TypeScript Port

1. **Timestamp Handling**: The Z-suffix conversion is critical — Python's isoformat() produces `+00:00`, but the code explicitly replaces it with `Z`.

2. **SHA256 Prefix**: Always use first 16 characters of hexdigest, not full digest.

3. **Deduplication Signature**: Messages use `(timestamp, role, content_hash)`, tool events use `(timestamp, summary_hash)`. The signature SET approach ensures first-occurrence preservation.

4. **Gemini's Recursive Traversal**: The visit function must properly handle deeply nested structures and update metadata at any depth level.

5. **Text Extraction**: Three different strategies (blocks, simple, gemini) — each handles different JSON structures with fallbacks.

6. **Client Inference**: Path-based detection with specific patterns; must handle path separators correctly.

7. **Encoding Handling**: Python uses UTF-8 with "ignore" error handling for JSONL; TypeScript should be flexible with malformed JSON.

8. **Type Coercion**: Heavy use of `str()` conversion — must replicate in TypeScript with proper null/undefined handling.

9. **Immutability**: Deduplication creates new lists; parsers return new dataclass instances.

10. **Normalization Flexibility**: `ensure_dict()` and `ensure_list()` provide safe defaults when types are wrong.
