- [HIGH] github-pages.ts:48 — worktreePath 使用 .git/gh-pages-init-worktree 作为路径：

- [HIGH] github-pages.ts:62 — git subtree push 是一个极其缓慢的操作，它会遍历整个 git 历史来提取子树。对于大型仓库或长历史，这可能耗时数分钟甚至更长，并且无超时控制（execSync 默认无超时）。应设置合理的超时，或改用 git worktree + git push --force 的方案。

- [MEDIUM] github-pages.ts:78-86 — makeRelative 函数：

  function makeRelative(base: string, target: string): string {
    const b = base.replace(/\\/g, '/').replace(/\/$/, '')
    const t = target.replace(/\\/g, '/').replace(/\/$/, '')
