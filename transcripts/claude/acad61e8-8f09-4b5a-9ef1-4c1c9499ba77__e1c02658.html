1. Client Type (Union)

2. TranscriptMessage Interface

interface TranscriptMessage {
  role: string              // Speaker role (e.g., "user", "assistant")
  text: string              // Message content
  timestamp: string | null  // ISO timestamp or null
}

Represents a single message in a conversation.

3. TranscriptToolEvent Interface

interface TranscriptToolEvent {
  summary: string           // Description of tool usage
  timestamp: string | null  // ISO timestamp or null
}

Represents tool/function calls during conversation.

4. ParsedTranscript Interface (Main normalized format)

interface ParsedTranscript {
  client: Client                    // Which AI platform
  session_id: string                // Unique session identifier
  title: string                     // Conversation title
  started_at: string                // ISO timestamp when started
  cwd: string                       // Working directory at time
  branch: string                    // Git branch at time
  messages: TranscriptMessage[]     // All conversation messages
  tool_events: TranscriptToolEvent[] // All tool invocations
  source_path: string               // Original file path
}

Normalized representation of parsed AI conversation, regardless of original client format.
