#### @openmnemo/cli (packages/cli/)
Entry point: dist/main.js (binary: openmnemo)
Build: tsup (uses tsup.config.ts)
Dependencies: @openmnemo/core, @openmnemo/sync, @openmnemo/types, commander@13.1.0

CLI Commands (from src/main.ts):

1. init - Initialize a MemoryTree workspace
- Options: --root, --project-name, --goal-summary, --locale, --date, --time, --force
- Handler: src/cmd-init.ts → cmdInit()

2. upgrade - Upgrade repo to MemoryTree format
- Options: --root, --project-name, --goal-summary, --locale, --date, --time, --format (text|json)
- Handler: src/cmd-upgrade.ts → cmdUpgrade()

3. import - Import one transcript file
- Options: --source (required), --root, --client (auto|codex|claude|gemini), --project-name, --global-root, --raw-upload-permission (not-set|approved|denied), --format (text|json)
- Handler: src/cmd-import.ts → cmdImport()

4. discover - Discover and import local AI transcripts
- Options: --root, --client (all|codex|claude|gemini), --scope (current-project|all-projects), --project-name, --global-root, --raw-upload-permission, --limit, --format
- Handler: src/cmd-discover.ts → cmdDiscover()

5. locale - Detect effective locale
- Options: --root, --locale, --format
- Handler: src/cmd-locale.ts → cmdLocale()

6. recall - On-demand sync and latest session recall
- Options: --root, --project-name, --global-root, --activation-time, --format
- Handler: src/cmd-recall.ts → cmdRecall()

7. daemon - Heartbeat lifecycle management
- Subcommands:
- install - Register with OS scheduler (options: --interval, --auto-push)
- uninstall - Remove scheduled task
- run-once - Execute single heartbeat cycle
- watch - Continuous loop (dev mode, option: --interval)
- status - Show heartbeat and lock state
- Handler: src/cmd-daemon.ts

Project utilities (src/project/):
- scaffold.ts - buildDatetime(), createMemoryDirs(), findExternalPolicySources(), resolveScaffoldPaths(), resolveTemplateDir(), scaffoldContentFiles(), writeTemplate()
- locale.ts - Locale detection
- upgrade.ts - Upgrade logic
