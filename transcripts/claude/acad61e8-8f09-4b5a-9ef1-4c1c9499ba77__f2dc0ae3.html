Exports:
- SearchResult interface: client, project, session_id, title, cwd, branch, started_at

### Key Design Patterns

1. Immutability: All functions return new objects, never mutate inputs
2. Deduplication: Messages and tool events deduplicated by content hash + timestamp
3. Multi-format Support: Parsers for Codex (JSONL), Claude (JSONL), Gemini (JSON/JSONL), Doubao (TXT)
4. Storage Abstraction: StorageAdapter interface enables local FS or Gitea API backends
5. Lazy Initialization: sql.js WASM module cached to avoid re-init
6. Posix Paths: All file paths normalized to forward slashes for cross-platform consistency
7. Manifest Tracking: Signatures track changes, preserve import timestamps, enable deduplication
