- project_slug: str — Project identifier
- raw_upload_permission: str — Permission level for raw transcripts

5. Partition by year/month from parsed.started_at or imported_at
6. Build 5 path structures:
- Repo: root/Memory/06_transcripts/{raw,clean,manifests}/CLIENT/YEAR/MONTH/ARTIFACT
- Global: global_root/{raw,clean}/CLIENT/PROJECT/YEAR/MONTH/ARTIFACT
- Global index: global_root/index/{manifests,search.sqlite,sessions.jsonl}
7. Create all parent directories
8. Copy source file to both repo_raw_path and global_raw_path
9. Build manifest dict (13 keys + 5 computed paths + counts)
10. Load existing global manifest; preserve import timestamp if unchanged
11. Write clean markdown to both locations
12. Write JSON manifests
13. Append to event log ONLY if manifest changed or didn't exist
14. Upsert SQLite search index
15. Return result dict with global_index_db added

- Input: int/float (Unix timestamp), datetime, or str (ISO8601)
- Output: ISO8601 string with Z suffix (UTC), no microseconds
- Handles: Z suffix conversion to +00:00, tzinfo conversion to UTC
- Fallback: current UTC time if unparseable

timestamp_partition(timestamp) -> tuple[str, str]
- Parse ISO8601 timestamp
- Return (year_str, month_str) from datetime.strftime("%Y"), strftime("%m")
- On parse error: return ("unknown", "unknown")

yaml_escape(value) -> str
- Escape: \ → \\, " → \", newline → \n

- If path doesn't exist: return None
- Read UTF-8, parse JSON
- On OSError/JSONDecodeError: return None
